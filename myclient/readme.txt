## Client Commands
- **register**: Register a new user.
- **login**: Log in and obtain an access token.
- **logout**: Log out and clear the access token.
- **list**: List all module instances.
- **view**: View all professors and their average ratings.
- **average**: View a specific professor's average rating for a specific module.
- **rate**: Submit a rating for a professor.
- **exit**: Exit the client program.
